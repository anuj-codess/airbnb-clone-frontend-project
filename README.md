# 🏠 Airbnb Clone (Frontend)

A responsive Airbnb clone frontend built using HTML, CSS, and JavaScript. This project replicates the modern UI of Airbnb with a clean and user-friendly design.

* Navigation bar with search functionality
* Clean layout and user-friendly interface
